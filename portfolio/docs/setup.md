# セットアップメモ

## 今回やったこと

- 開発サーバーが 5173 に出ない問題を調べた。
- 原因は、Vite が 5173 を使えないと別ポートへ自動退避していたことだった。
- そのため、Vite の設定を 0.0.0.0:5173 に固定し、strictPort を有効にした。
- 起動は workspace ルートではなく、portfolio ディレクトリで行う。

## 変更内容

- portfolio/vite.config.ts に server 設定を追加した。
- host を 0.0.0.0 にした。
- port を 5173 に固定した。
- strictPort を true にした。

## 起動手順

1. portfolio ディレクトリに移動する。
2. npm run dev を実行する。
3. http://localhost:5173 を開く。

## もし表示されないとき

- まず 5173 を別のプロセスが使っていないか確認する。
- 5173 が使われている場合は、そのプロセスを止めてから再起動する。
- ワークスペース直下で npm run dev を実行しない。
